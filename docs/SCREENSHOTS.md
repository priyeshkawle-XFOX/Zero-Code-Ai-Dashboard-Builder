## 📐 Screenshot Guidelines

When capturing screenshots, please follow these guidelines:

### 🎨 Composition

- Use a clean browser window (hide bookmarks bar, extensions)
- Capture at standard resolutions (1920x1080 for desktop, 375x812 for mobile)
- Use realistic data in the dashboard

### 🖼️ Format

- **Format:** PNG (preferred) or JPG
- **Quality:** High resolution, no compression artifacts
- **Naming:** Use kebab-case (e.g., `dashboard-line-chart.png`)

### 🌈 Theme

- Capture both light and dark mode versions
- Use consistent color schemes
- Ensure good contrast and readability

---

## 🛠️ How to Take Screenshots

### Windows

1. **Full Screen:** `Win + PrtScn`
2. **Window:** `Alt + PrtScn`
3. **Snipping Tool:** `Win + Shift + S`

### macOS

1. **Full Screen:** `Cmd + Shift + 3`
2. **Selection:** `Cmd + Shift + 4`
3. **Window:** `Cmd + Shift + 4 + Space`

### Linux

1. **GNOME Screenshot:** `PrtScn`
2. **Flameshot:** `flameshot gui`
3. **Scrot:** `scrot -u`
